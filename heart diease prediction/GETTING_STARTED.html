Getting Started with Heart Disease Prediction Application

This guide will help you set up and run the Heart Disease Prediction web application on your local machine.

Note: This application requires Python 3.7 or higher.

Installation Steps

Step 1: Install Required Dependencies

Open a command prompt or terminal and navigate to the project directory. Then run:

pip install -r requirements.txt

This will install all the necessary Python packages for the application.

Step 2: Run the Application

From the project directory, run:

python run.py

The first time you run the application, it will train the machine learning model, which may take a few minutes. Subsequent runs will use the saved model.

Step 3: Access the Web Interface

Once the application is running, open your web browser and navigate to:

http://localhost:5000

You should see the Heart Disease Prediction Tool interface.

Application Features

  • Prediction Tool: Enter your health information to get a personalized heart disease risk assessment
  • Dashboard: View key statistics and insights from the dataset
  • Data Insights: Explore visualizations showing relationships between various risk factors and heart disease
  • History: Review your previous predictions and track changes in your risk profile
  • About: Learn more about the project, the dataset, and heart disease risk factors

Troubleshooting

Common Issues

Port Already in Use

If you see an error like "Address already in use", it means port 5000 is already being used by another application. You can modify the port in run.py by changing:

app.run(debug=True)

to:

app.run(debug=True, port=5001)

Missing Visualizations

If visualizations are not appearing, you can regenerate them by visiting:

http://localhost:5000/generate_visualizations

Model Training Errors

If you encounter errors during model training, ensure you have the correct version of scikit-learn installed and that the dataset file train_updated.csv is in the project directory.

  2. Explore the dashboard to understand the dataset statistics
